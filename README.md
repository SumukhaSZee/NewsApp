# News App 

**Frameworks:**
>  MVVM<br>
>  Retrofit<br>
>  Coroutines<br>
>  Navigation Components.<br>
<br>

Functionalities:
<br>
Displays current breaking news.<br>
Search option to search for news articles<br>
Displays the news articles in WebView<br>
